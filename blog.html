Question No: 1

Question: Discuss the scope of var, let, and const

Answer: In Javascript var, let and const are used to declare a variable. They are unique in their own way. These 3 has their own scope.

Var: Var is a function scope but It can hoist. So, when a variable is declared using Var, it hoists that variable to the top and makes it a global variable. Var can also be re-declared and re-assigned.
Let: variable that are declared using Let are block scoped. Which means those variables can not be accessed outside of the block. Let doesn't hoist, so unlike var, it doesn't turn it's variable into a global variable.
Const: Like Let, variables that are declared using Const is also a block scoped. Const variables aren't accessable outside of the block. The value of Const can not be re-assigned. Also, It doesn't hoist it's variable to the top.

Question: Tell us the use cases of null and undefined

Answer: Undefined and Null both refers to a missing value. Usually Undefined occurs when a declared but not initialized varible is being used. If we try to access an object property or an array element that doesn't exist also gives us undefined.
On the other hand, Null! is used to show the absence of a value or empty value. Unlike undefined, Null is used to showcase an empty value or clear the value of a property.

Question: What do you mean by REST API?

Answer: Representational State Transfer, or REST, is an architectural design approach for creating networked applications. A web service that complies to the rules and guidelines of REST is known as a RESTful API (REST API). REST is an architectural set of concepts and limitations, not a protocol, that offers a simple and scalable approach to designing networked applications.
RESTful APIs are frequently used to create web services that can run on a variety of hardware and operating systems because they are easy to use, scalable, and capable of operating over HTTP. They are often employed in the development of web applications, the backends of mobile applications, and the online integration of many systems and services.
